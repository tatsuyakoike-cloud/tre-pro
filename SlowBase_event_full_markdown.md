# SlowBase event

## SERAMADE支援を起点に、売上につながる型をつくる

---

## このページで伝えたいこと

今回のSERAMADE支援は、単なるイベント出展ではありません。

餅田農園にとっては、  
SERAMADEの売上を伸ばすための機会です。

SlowBaseにとっては、  
撮影だけに頼らない仕事をつくるための実証です。

まずは小さく始めて、  
売上が出るか、定期購入につながるか、毎月続けられる形にできるかを見ていきます。

その結果をもとに、  
餅田農園に対してイベント出展費用や運営支援費をいただける状態を作りたいです。

---

## 1. まず結論

今のSlowBaseにとって、今回のSERAMADE支援は進める価値があります。

理由は、既存顧客である餅田農園の売上を伸ばしながら、  
SlowBaseとしても新しい仕事の形を作れる可能性があるためです。

これまでSlowBaseでは、ドローン撮影や映像制作を入口にした仕事が中心にありました。

ただ、撮影案件は常に安定して発生するものではありません。  
また、撮影をしたからといって、必ずその後の運用支援や追加案件につながるとも限りません。

そのため、これからは撮影だけではなく、  
商品やブランドの売上に近いところまで関わる仕事を作っていきたいです。

今回のSERAMADE支援は、そのための実証案件です。

---

## 2. SERAMADEについて

SERAMADEは、餅田農園が展開しているブランドです。

自然由来の商品、地域性、農業、健康、美容といった要素を持っています。

SlowBaseが今後広げていきたい、  
自然・地域・農業・体験などを含む支援領域とも相性があります。

ここで大事なのは、SERAMADEの商品をただ紹介することではありません。

イベントに来た人に、

- 何の商品なのか
- どんな人に合うのか
- なぜ良いのか
- どこで買えるのか
- 続けるとどう良いのか

をわかりやすく伝えることです。

そこまで設計できれば、  
単なる出展ではなく、売上につながる導線になります。

Walklog社が運営する「よりみちYoga」というイベントに、  
SERAMADEの商品を出させていただける流れができています。

- 7月25日にSERAMADEのドリンクを設置
- 8月22日も設置予定

- 毎回50人前後が来場するイベント

## 4. この施策でやりたいこと

やりたいことは、イベントに商品を置くことではありません。

やりたいのは、  
SERAMADEが売れる流れを作ることです。

流れとしては、以下です。

1. イベントでSERAMADEを知ってもらう
2. 試飲やPOPで商品の魅力を伝える
3. QRコードでECやInstagramへ誘導する
4. 単体購入につなげる
5. できれば定期購入につなげる
6. 結果を見て次回の見せ方を改善する

ここまでできれば、  
イベント出展ではなく、販売導線の実証になります。

## 5. 餅田農園としての売上シミュレーション

SERAMADEのドリンクは1本4,000円です。

仮に月1回イベントに出るとして、  
販売本数ごとの売上を整理すると以下のようになります。

| 1回あたりの販売本数 | 1回あたり売上 | 年間売上（月1回・12回） |
|---:|---:|---:|
| 5本 | 20,000円 | 240,000円 |
| 10本 | 40,000円 | 480,000円 |
| 20本 | 80,000円 | 960,000円 |
| 50本 | 200,000円 | 2,400,000円 |

まずは5本でも売れれば、1回で20,000円です。  
月1回続けば、年間で240,000円になります。

10本売れれば年間480,000円。  
20本売れれば年間960,000円。  
50本売れれば年間2,400,000円です。

最初から50本を狙うというより、  
まずは5本、次に10本、そこから20本という形で、  
毎回改善していくのが現実的です。

ただ、毎回50人前後が来るイベントで、  
最終的に50本売れる状態まで作れれば、  
餅田農園にとってかなり大きな販売機会になります。

## 7. 定期購入が取れると、もっと強い

この施策で大事なのは、当日売上だけではありません。

もし定期購入が取れれば、売上は積み上がります。

SERAMADEのドリンクが月1本購入されるとすると、  
1人あたりの年間売上は以下です。

```text
4,000円 × 12ヶ月 = 48,000円
```

つまり、定期購入者が1人増えるだけで、  
年間48,000円の売上になります。

イベントごとに1人でも定期購入者が増えれば、  
その人だけで年間48,000円の売上です。

毎月1人ずつ定期購入者が増えれば、  
イベント当日の販売以上に、餅田農園側の売上インパクトは大きくなります。

だからこそ、この施策は  
その日に何本売れたかだけで判断しない方が良いです。

見るべきなのは、  
継続購入につながるお客様を作れるかどうかです。

## 10. 売上が見えれば、SlowBaseの仕事にもなる

最初は実証として進めます。

ただし、イベントを通じて餅田農園側の売上が上がることを見せられれば、  
SlowBaseとしても、イベント出展に関わる費用をいただきやすくなります。

たとえば、以下のような費用です。

- イベント出展運営費
- チラシ制作費
- POP制作費
- QR導線設計費
- SNS告知文作成費
- イベント後レポート作成費
- EC導線改善費
- 月次運用費

大事なのは、最初から大きく請求することではありません。

まずは餅田農園側にとって、  
「このイベントに出ると売上が上がる」  
という状態を作ることです。

仮に毎回50人前後が来るイベントで、  
10本、20本、最終的に50本売れるようになれば、  
餅田農園としてもこのイベントに出る意味がはっきりします。

50本売れれば、1回あたり200,000円の売上です。  
月1回続けば、年間2,400,000円です。

さらに定期購入まで取れれば、売上はもっと積み上がります。

その状態を作れれば、  
SlowBaseに支払っている費用も、餅田農園側として回収しやすくなります。

だからこそ、売上が見えた段階で、  
イベント出展費用や運営支援費として、追加で費用をいただく流れを作りたいです。

これらは撮影案件がなくても受けられる仕事です。

これは、SlowBaseが今後やりたい方向とかなり近いです。

撮影だけではなく、

## 12. 毎回見る数字

この施策では、感覚だけで進めないようにします。

毎回見るべき数字は以下です。

| 項目 | 見る理由 |
|---|---|
| 来場者数 | 何人に接点を作れたかを見る |
| 試飲数 | 商品に興味を持ってもらえた数を見る |
| 販売本数 | その場の売上を見る |
| 売上金額 | 餅田農園側への売上貢献を見る |
| QRコード読み込み数 | ECやInstagramへの導線が機能したかを見る |
| Instagramフォロー数 | 継続接点が作れたかを見る |
| EC流入数 | イベントから購入ページへ送れたかを見る |
| 定期購入件数 | 継続売上につながったかを見る |
| イベント後の問い合わせ数 | 後追いの反応を見る |
| 次回購入の見込み | 次回改善の材料にする |

これらを毎回記録すれば、  
なんとなく出展したではなく、  
売上につながるイベント施策として改善できます。

餅田農園に対しても、  
今回いくら売れたかだけではなく、  
次回どうすればもっと売れそうかまで返せます。

ここまでできれば、SlowBaseの仕事は単なる手伝いではなく、  
売上を作るための販促運用支援になります。

## 15. この施策のリスク

もちろん、良いことだけではありません。

リスクもあります。

- 初回から大きく売れるとは限らない
- SERAMADE単体では継続単価が小さい可能性がある
- Walklog側の外注ニーズはまだ確定していない
- EC領域は樋高さん側にも知見があるため、SlowBaseの役割を明確にする必要がある
- 小池が入りすぎると属人的になる
- 西さんに任せるにはマニュアル化が必要

なので、最初から大きな売上を期待しすぎるのではなく、  
売上につながる型を作れるかを見る施策として進めたいです。

## 16. 会社として進める意味

今回の施策には、会社として進める意味があります。

### 1. 撮影以外の仕事を作れる

撮影案件が動かない時期でも、  
販促物、SNS、導線設計、イベント運用で仕事を作れます。

### 2. 既存顧客から始められる

完全な新規開拓ではないため、  
小さく始めて検証できます。

### 3. 顧客の売上に近いところで関われる

SERAMADEの販売本数や定期購入数で、  
成果が見えやすいです。

### 4. 月次運用にしやすい

イベントが毎月あるため、  
単発ではなく継続支援に変えやすいです。

### 5. Walklog側の仕事に広がる可能性がある

イベント運営、販促、EC導線など、  
SERAMADE以外の仕事に広がる可能性があります。

### 6. SlowBaseの方向性と合っている

自然、地域、健康、農業、体験を扱うため、  
今後のネイチャー産業支援の実績にしやすいです。

## 17. 最終的に作りたい形

最終的には、以下のような流れを作りたいです。

1. SERAMADEをイベントに出す
2. チラシ、POP、QR導線を整える
3. イベントで売上を作る
4. 定期購入につなげる
5. 餅田農園側の売上が上がる
6. SlowBaseの支援価値が見える
7. イベント出展費用や運営費をいただける
8. Walklog側のEC・販促支援にも広げる
9. 他の自然・地域・健康系ブランドにも横展開する

この流れが作れれば、  
SlowBaseとして新しい収益メニューになります。

## 18. 最後に

今回のSERAMADE支援は、単なるイベント出展ではありません。

餅田農園にとっては、  
SERAMADEの売上を上げる機会です。

SlowBaseにとっては、  
撮影以外の仕事を作る実証です。

Walklogとの関係では、  
イベント運営やEC導線の支援に広がる可能性があります。

まずは7月25日と8月22日のイベントで、  
販売本数、QR読み込み、EC流入、定期購入を見ます。

5本売れれば2万円。  
10本売れれば4万円。  
20本売れれば8万円。  
50本売れれば20万円。

月1回続けば、年間では24万円〜240万円の売上インパクトになります。  
さらに定期購入が取れれば、売上はもっと積み上がります。

まずは餅田農園の売上を上げる。  
その実績をもとに、SlowBaseの支援価値を見せる。  
そのうえで、イベント出展費用や月次運用費をいただく。

この流れが作れれば、  
餅田農園にとっても、SlowBaseにとっても良い仕事になります。

そのため、今回のSERAMADE支援は、  
SlowBaseの新しい事業モデルを作るための実証案件として進めたいです。
