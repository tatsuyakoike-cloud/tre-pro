# トレプロ営業方針改善計画

## 1. 全体方針

今後の営業活動は、**オフライン営業**と**オンライン営業**の2軸で進める。

| 領域 | 役割 | 目的 |
|---|---|---|
| オフライン営業 | 代理店・紹介・地方営業・飛び込み・訪問・クロージング | 短期で商談と受注を作る |
| オンライン営業 | LP経由リストの整理・掘り起こし・配信・架電・ナーチャリング | 既存リストを営業資産化し、再現性のある商談創出を作る |

この2軸で進める理由は、LP広告の獲得効率が悪化している中で、広告単体の改善だけに依存せず、**短期の受注創出**と**中長期の営業基盤づくり**を同時に進める必要があるためである。

7月・8月は、Q4に向けて以下の状態を作る。

| 項目 | 8月末時点で作る状態 |
|---|---|
| 営業体制 | オフライン営業とオンライン営業の役割が明確になっている |
| LP経由 | 既存リストから商談・受注が生まれている |
| 数値管理 | 成約数、LTV合計、総獲得コスト、LTV/CAC倍率を確認できる |
| リスト管理 | 確度別・Tier別に営業対象が整理されている |
| ナーチャリング | メール、ホワイトペーパー、ウェビナー、LINE等で継続接点を作れている |
| Q4準備 | 9月以降、CRM・SFA・MAへ接続できる運用ルールが整理されている |

---

## 2. 現状認識

### 2-1. 現状の課題

| 課題 | 内容 |
|---|---|
| LP経由の獲得コスト悪化 | 広告費をかけても、以前より効率よく顧客獲得できていない |
| LTVの低下 | 顧客の継続期間が短くなり、1社あたりの将来売上が下がっている |
| 広告依存の限界 | 広告改善だけで短期的に大きく回復する見込みが立ちづらい |
| 既存リストの未活用 | 過去接点・商談履歴がある企業を十分に掘り起こせていない |
| 営業ルールの未整備 | 確度、優先順位、失注理由、次回アクションの管理が統一されていない |

### 2-2. 今回の対応方針

- オフライン営業では、代理店・紹介・地方営業・飛び込みを活用し、短期の受注機会を作る。
- オンライン営業では、LP経由の既存リストを整理し、掘り起こし施策を新たに企画・実行する。
- これまでの商談受注率を前提に置きすぎず、今回の掘り起こし施策で新しい基準値を作る。
- 施策ごとに、成約数・LTV合計・総獲得コスト・LTV/CAC倍率を確認する。

## 3. LP経由の数値前提

### 3-1. 数値の位置づけ

以下の数字は、現時点での**仮置き**である。  
確定値ではなく、LP経由施策を継続判断できる水準まで戻すための、シミュレーション上の目標値として扱う。

今回の計算は、**税抜概算ベース**で統一する。  
参考として、税込ベースでは総コストは約790〜800万円となるが、資料内のシミュレーションでは税抜概算の約700万円を基準にする。

| 項目 | 数値 | 説明 |
|---|---:|---|
| 総獲得コスト | 約700万円 | LP経由施策の税抜概算コスト。広告費・営業人件費・外注費等を含む |
| 判断基準 | LTV/CAC 3.0倍以上 | 継続判断の最低ライン |
| 必要LTV | 2,100万円 | 700万円 × 3.0倍 |
| 1社あたりLTV目安 | 330万円 | 受注1社あたりの期待LTV |
| 最低必要成約数 | 7社 | 2,100万円 ÷ 330万円 ≒ 6.36社 → 切り上げ |

### 3-2. LTV・CAC・ユニットエコノミクスの整理

| 指標 | 定義 | 今回の扱い |
|---|---|---|
| CAC | 顧客獲得にかかったコスト | LP経由施策の税抜概算総コストとして約700万円で置く |
| LTV | 顧客から将来的に得られる売上・価値 | 受注1社あたり330万円を目安に置く |
| ユニットエコノミクス | LTV ÷ CAC | LP経由施策全体では「LTV合計 ÷ 総獲得コスト」で見る |

### 3-3. 補足

- 「総獲得コスト 約700万円」は、LP経由施策全体にかかる税抜概算コストとして扱う。
- 税込ベースで保守的に見る場合、総コストは約790〜800万円となる。
- 税込800万円ベースで見る場合は、必要LTVは2,400万円、必要成約数は8社となる。
- 資料上では、税抜概算ベースの約700万円に統一し、最低7社成約を基準に管理する。
- 今後、実績が出次第、総獲得コスト・LTV・必要成約数は更新する。
